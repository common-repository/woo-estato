== Description ==

Its an addon for <a href="https://wordpress.org/plugins/real-estate-manager/">Real Estate Manager</a> to let you manage paid listing packages for agents on your site. It also requires WooCommerce to work, as it uses the payment methods and processing system available in the WooCommerce plugin.
